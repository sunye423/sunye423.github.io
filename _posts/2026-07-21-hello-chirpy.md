欢迎来到我的个人技术博客。

这个网站使用 Jekyll 生成静态页面，采用 Chirpy 主题，并通过 GitHub Actions 自动部署到 GitHub Pages。

## 为什么搭建这个博客

我希望有一个长期积累技术内容的地方，用来记录：

- 编程与项目实践
- 开发工具的使用经验
- 问题排查过程
- 学习笔记与技术总结

## 后续计划

后续会持续整理和发布技术文章，让实践经验能够被检索、复用和分享。

博客已经启程，保持记录，持续成长。
